== Description ==

WP Mail SMTP- WP 4.4 Support, works with Gmail,Godaddy, Yahoo,Hotmail and any other mail provider.
Use external SMTP service for your outgoing mail.


== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `ex_smtp_mail` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
